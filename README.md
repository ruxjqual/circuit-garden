# Puzzle Arcade

A static HTML game hub for Cloudflare Pages.

## Games

- `circuit-garden/` - A bilingual circuit-rotation puzzle.
- `number-game/` - A sliding number puzzle.
- `jigsaw-20/` - A 20 x 20 picture puzzle.

## Deploy

- Project type: static site
- Build command: `exit 0`
- Build output directory: `/`
